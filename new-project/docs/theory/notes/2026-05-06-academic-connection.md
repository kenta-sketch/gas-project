# クアッドマインド統合モデルの学術的接続

## エグゼクティブサマリ

A/B/C/Dを既存心理学・神経科学へ厳密に「翻訳」する**中位統合理論**として再定義。結論:

| 要素 | 学術的位置づけ |
|---|---|
| **A** | 身体化された情動サリエンス |
| **B** | 社会評価に基づく自己意識的感情 |
| **C** | 経験が圧縮された直観的判断 |
| **D** | 熟慮的認知制御 |

A/B/C/Dは単一の脳部位ではなく、理論・課題・神経ネットワークを横断する潜在機能。感情エピソードの時間展開の中で相互作用する。

> クアッドマインドは既存理論の代替ではなく、**既存理論を現場適用可能な四象限に圧縮した操作モデル**として扱うべき。

## 各エンジンの学術的接続

### A: 身体化された情動サリエンス
- **既存理論**: 基本感情(Ekman)、core affect、Jamesian身体フィードバック、BIS/BAS
- **神経基盤**: 扁桃体・前部島皮質・腹側線条体・自律神経系
- **測定尺度**: PANAS、BIS/BAS、状態不安、心拍・皮膚電気
- **代表研究**: 恐怖条件づけ、報酬予期 fMRI、BIS/BAS予期実験
- **介入**: 脅威手掛かりの削減、睡眠・呼吸・休息の最適化、第一印象設計

### B: 社会評価に基づく自己意識的感情
- **既存理論**: 評価理論(Lazarus, Scherer)、自己意識的感情、社会的評価脅威、表示規則
- **神経基盤**: mPFC・TPJ・ACC・島皮質・扁桃体
- **測定尺度**: SFNE、TOSCA-3、日本語TEIQue-SF、社会不安指標
- **代表研究**: TSST、Cyberball、恥・罪悪感の神経レビュー
- **介入**: 心理的安全性の設計、恥ではなく修正可能性の提示、信頼形成

### C: 経験依存的直観
- **既存理論**: ソマティック・マーカー(Damasio)、自然主義的意思決定、経験的システム、熟達したSystem 1
- **神経基盤**: vmPFC/OFC・線条体・海馬・島皮質
- **測定尺度**: IPSI直観性、Iowa Gambling Task、経験年数、シナリオ判断
- **代表研究**: IGT、直観的熟達の境界条件研究
- **介入**: ケース反復、即時フィードバック、アフターアクションレビュー

### D: 熟慮的認知制御
- **既存理論**: System 2(Kahneman)、実行機能、認知制御、認知的再評価(Gross)
- **神経基盤**: dlPFC・vlPFC・dACC・前頭頭頂制御ネットワーク
- **測定尺度**: 認知欲求尺度、CRT、BRIEF-A、BFI-2-J
- **代表研究**: 再評価メタ分析、PFCの統合理論、BRIEF-A妥当化
- **介入**: チェックリスト、プレモータム、再評価訓練、説明責任の明示

## 統合枠組み

> 情動は「まず起こり、後で考える」だけでも、「まず考えてから起こる」だけでもなく、
> A/B/C/Dが循環しながらエピソードごとに構成される。

```
[刺激・状況]
  ├─→ A: 身体化された情動サリエンス
  ├─→ B: 社会評価と規範感情
  ↓
[過去経験・熟達記憶] ──→ C: 経験依存的直観
[目標・規則・説明責任] ──→ D: 熟慮的認知制御
  ↓
A → C / B → D / C → 行動 / D → 行動 / D → A / D → B
  ↓
[結果・身体反応・他者反応] ──→ A/B/C にフィードバック
```

## 既存理論の系譜

- **基本感情論(Ekman)**: Aの高速で比較的普遍的な表出を支える
- **評価理論(Lazarus, Scherer)**: 出来事の評価が行動傾向・生理反応・主観感情・ラベリングへ展開する過程
- **身体フィードバック(James)・二要因説(Schachter)**: Aの生理的覚醒とB/Dの意味づけをつなぐ
- **構成主義(Barrett)**: A=core affect・内受容感覚、B/D=概念化・社会的カテゴリー化、C=過去経験に基づく予測として再編成
- **二重過程論(Kahneman)**: A/Cの高速処理とDの低速処理の差
- **脅威研究(LeDoux)**: Aの防御的サリエンス
- **情動制御研究(Gross)**: Dの再評価・注意制御を補強

> 二重過程論は「二つの脳」ではなく、処理特性の区別。
> Bは社会的ヒューリスティクスとしてSystem 1にも、規範的自己制御としてSystem 2にもまたがる。
> A/B/C/Dは脳内の四箱ではなく、**身体・社会・経験・制御という四つの説明軸**である。

## 学術的弱点(誠実な開示)

- A/B/C/Dは標準化された学術分類ではない(現時点)
- Cは「直観への嗜好」「直観の正確さ」「領域熟達」が混ざりやすい
- Bは社会不安・恥傾向・承認欲求・trait EIが部分的に重なる
- Aは状態依存性が高い

→ 単純な特性検査として使うのではなく、**状態×特性×状況適合の三層**で扱う必要がある。

## 倫理配慮(運用時の禁止事項)

- 人事選抜でA/Bを欠陥のように固定化して使用しない
- 教育で恥を利用して行動修正しない
- 営業でA/Bを過剰刺激して恐怖や社会的不安を利用しない

> 推奨運用: **選別ではなく発達支援、単発診断ではなく継続的フィードバック、個人責任化ではなく環境調整との併用**

文化差の大きい表示規則や社会的評価感受性を踏まえ、ローカル規範化と説明可能性を担保した上で運用。
