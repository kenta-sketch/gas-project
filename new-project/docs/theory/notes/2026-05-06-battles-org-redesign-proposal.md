# BRAIN JUICE × 業務委託 組織再設計提案書(CONFIDENTIAL)

> 離職率50%(半年) → 25%以下へ / 組織効率1.9倍

クアッドマインド理論 × 心理的安全設計 × 仕組み化マネジメント の統合提案書。

## 経営判断サマリー

### Q1. いくら儲かるか
新規売上を直接創出するものではない。**既存人材の収益効率を構造的に引き上げ、現在取りこぼしている利益を回収するモデル**。

| 指標 | 現状 | 改善後 | 差分 |
|---|---|---|---|
| 稼働人数(半年後残存) | 7-8名 | 11-12名 | +4名 |
| 1人あたり生産性 | ×1.0 | ×1.25 | +25% |
| 組織全体生産性 | 7.5 | 14.4 | **約1.9倍** |
| 採用・教育コスト | 高 | 低 | 削減 |

### Q2. いつ回収できるか
- **1ヶ月以内**: 幹部の発言質変化・現場の相談量増加・心理指標可視化
- **3ヶ月以内**: 離職率の鈍化・数値パフォーマンス安定・個人差縮小
- **6ヶ月以降**: 離職率50%→25%達成・効率1.9倍・仕組み自走化

## 1. 現状分析・構造診断

### 1.1 離職の再現構造

> 離職は偶発ではなく、**採用・配属・評価設計の分断によって再現性をもって量産されている**。

#### 離職の連鎖メカニズム
1. **不適合感**: 採用時期待と現実の乖離
2. **無意味感**: 現業務が将来に接続していない認識
3. **成長欠如感**: 努力と成果の不結合
4. **合理的離脱**: 上記3条件揃った時点で合理的選択に

> 解決すべきは個人ではなく構造。個人を変えようとするアプローチは根本的に機能しない。

### 1.2 携帯販売業の再定義

| 一般的な誤解 | 本質的定義 |
|---|---|
| スクリプト通り話せばいい | BtoC×BtoB同時訓練環境 |
| 商品説明が主業務 | 自己価値の翻訳・伝達能力 |
| 誰でもできる単純作業 | 相手の言語に合わせる適応力 |
| キャリアに繋がらない | あらゆる市場で転用可能な基礎能力 |

## 2. 基本思想

### 営業の再定義

> **営業とは「自己価値を伝達する技術」である**

商品を売る行為ではなく、自己価値を翻訳し伝達する技術。これにより営業スキルは特定業界の業務ではなく、**あらゆる領域に転用可能な普遍的基礎能力**となる。

### 四要素統合設計

| 要素 | 内容 |
|---|---|
| **意味** | 初日に確定。現業務と未来を接続 |
| **成長** | プロセスを分解し数値化。成約ゼロ期間でも成長を可視化 |
| **関係** | 三層構造(個別・横・縦)で孤立を構造的に消滅 |
| **報酬** | 短期(行動促進)と長期(継続正当化)を分離設計 |

## 3. クアッドマインド × マネジメント

### 4象限の指導アプローチ

| 象限 | 有効な指導 | 禁止事項 | 離職リスク要因 |
|---|---|---|---|
| **A**(動物的感情) | 安心感の提供・小さな成功体験 | 叱責・プレッシャー・他者比較 | 恐怖・不安の放置 |
| **B**(条件化感情) | 承認・感謝・関係性の質維持 | 無視・冷淡な評価・孤立 | 承認の枯渇 |
| **C**(動物的理性) | 権限移譲・自由度・スピード | 過剰管理・細かいルール | 裁量のなさ |
| **D**(機械的理性) | 明確なKPI・論理的フィードバック | 感情論・曖昧な評価 | 根拠のない指示 |

### クアッドビジョン × 未来解釈

| 解釈型 | 特徴 | アプローチ |
|---|---|---|
| 楽観的ポジティブ | 自走型・チャレンジ志向 | 過度な期待管理 |
| 楽観的ネガティブ | 楽観だが失敗リスク回避なし | 現実的シミュレーション |
| 悲観的ポジティブ | 慎重だが行動量確保 | 小さな成功体験 |
| **悲観的ネガティブ** | **最高離職リスク**(不安・諦め共存) | 意味の再接続と安全な場の設計が緊急 |

## 4. 幹部配置・ローテーション

### 配置の基本ロジック
クアッドマインドスコアと実性格評価(傾聴力・安心感付与・指示力・決断力)の2軸で幹部を分類し、人材フェーズに応じた最適担当者を動的に決定。

### 幹部2タイプ

#### 共感型幹部(Support型)
- 配置条件: A/Bスコア高・傾聴力高
- 主担当: 入社0-3ヶ月
- KPI: 離職率25%以下 / 1on1実施率100% / 継続意志4.0以上 / 行動実行率70%以上
- NG: 数字での詰め・他者比較指導・正論優先

#### 管理型幹部(Control型)
- 配置条件: C/Dスコア高・指示力明確
- 主担当: 入社3-6ヶ月
- KPI: 成約率向上 / 行動量安定 / 売上達成
- NG: 感情優先指導・曖昧な判断・指示不足

### 1年間ローテーションモデル

| フェーズ | 期間 | 担当型 | 目的 |
|---|---|---|---|
| 1 | 0-3ヶ月 | 共感型 | 離職防止・行動習慣・心理安定 |
| 2 | 3-6ヶ月 | 管理型 | 数値強化・成果安定・再現性 |
| 3 | 6-9ヶ月 | 逆ローテーション | 苦手領域補完・統合思考 |
| 4 | 9-12ヶ月 | 統合運用 | 自走化・幹部候補化 |

### 引き継ぎプロトコル
> 「人材ではなく状態を引き継ぐ」 — クアッド状態・強み弱み・成功パターン・NGパターンをセットで渡す。15分口頭+議事録共有が標準。

## 5. 1on1運用設計

### 1on1の再定義

> 1on1は進捗確認ではない。**心理状態を定量化し、行動変容に接続するための運用装置**である。

### 初月フェーズ(週別)

| 週 | フェーズ | 目的 | 実施内容 | 重要原則 |
|---|---|---|---|---|
| 1週 | 承認 | 不安解消・心理安全 | 感情ヒアリング・承認・意味説明 | 「正しさ」より「安心」 |
| 2週 | インプット | 基礎知識定着 | 商材理解・トーク型・音声活用 | 「使える」状態にする |
| 3週 | 関係構築 | 立ち位置確立 | 立ち回り指導・BtoB意識 | 「売る前に環境を整える」 |
| 4週 | 自己理解 | 特性理解 | クアッドマインド分析・ビジョン | 「どう動くか」を理解 |

### 1on1運用ルール(固定プロトコル)

| 項目 | 基準 |
|---|---|
| 頻度 | 2週間に1回(固定) |
| 実施者 | 直属上司ではなく担当幹部 |
| 場の定義 | 評価ではなく安全な開示空間 |
| 記録 | 全件議事録化(必須) |
| 改善指示 | **1点のみ**(複数禁止) |
| 対話スタンス | 否定せず・遮らず・決めつけず |

## 6. 生産性向上モジュール

### 商材別資料 + 音声設計

| 1枚資料 | 3分音声 |
|---|---|
| 結論(何が得か) / ターゲット属性 / 一言価値 / NGパターン | 言い回し・間 / トーク流れ / 詰まり対処 / 成約実例 |

> 「覚える資料」ではなく「現場で使うツール」

### 本質営業の統一フロー(5ステップ)

1. **状況把握** - 現状・利用環境・不満をオープンクエスチョンで
2. **感情把握** - 言葉裏の感情・優先事項・決定権者を特定
3. **価値接続** - 顧客状況と商材ベネフィットを生活レベルで接続
4. **提案** - スペックではなく「生活がどう変わるか」
5. **納得形成** - 購入理由を顧客自身の言葉で確認

### 「めんどくさい」の払拭設計
- 1日1アクション固定
- 成果ではなく実行を評価
- 音声で即行動できる状態の維持

## 7. 離職時ヒアリング

### 離職の再定義

> 離職 = 失敗 ではない。**離職 = 構造の不具合が顕在化した状態**。

> 離職は「最も精度の高い組織データが取得できるタイミング」。在籍中は忖度・評価バイアスがあるが、離職時は評価から解放され、最も正確な情報が取得可能。

### 固定ヒアリングフォーマット(5項目)

1. 「いつ頃から違和感がありましたか?」 → 問題発生タイミング
2. 「最終的に辞めようと思った理由は?」 → トリガー要因
3. 「何があれば続けていましたか?」 → 改善ポイント
4. 「同じ条件の人が入社したら、どうなると思いますか?」 → 個人 vs 構造の切り分け
5. 「それは事実ですか?印象ですか?」 → データ精度向上

### データ管理
意味・成長・関係・報酬の4分類で整理。月次で共通パターン・発生タイミング・役職偏りを分析。**取って終わりにしないことが本施策の生命線**。

## 8. 心理的安全設計

### 三層接続

| 接続 | 役割 |
|---|---|
| **個別**(1on1) | 心理状態の把握・不安の言語化・意味の接続 |
| **横**(共有会) | 比較と共感を生む。「自分だけじゃない」 |
| **縦**(上司連携) | 組織との接点維持。存在認識の確信 |

> 孤独は感情ではなく構造である。三層が同時に機能した時、孤立は消滅し、離職の合理性が消える。

### 意味設計(初日確定プロトコル)

| 現業務 | 変換される能力 | 将来活用領域 |
|---|---|---|
| 営業 | 収益化能力 | クリエイター・フリーランス・事業主 |
| 接客 | ファン形成能力 | ブランド構築・SNS・コミュニティ |
| 提案 | 表現能力 | プレゼン・企画・クリエイティブ |
| 交渉 | 価値翻訳能力 | 市場での自己表現・コミュニケーション |

### 成長の可視化(売上以外の評価対象)

| プロセス | 計測 | 基準 |
|---|---|---|
| ヒアリング精度 | 1on1ログ+同行評価 | 週次3.5以上 |
| 提案成功率 | 提案÷成約 | 月次改善トレンド |
| 会話継続力 | 平均接客時間 | 3分以上 |
| クロージング率 | 試行÷成約 | 月次トレンド |
| 再現性 | フロー実行率 | 80%以上 |

## 9. ROI試算

### 工数(幹部1人あたり月間)
- 1on1(10名担当): 10時間
- ヒアリング分析: 発生分
- 月次共有会: 1.5時間
- ノウハウ収集: 2時間
- スコア分析・レポート: 3時間
- **合計: 約16.5時間/月**

### 効果
- 残存人数: 7.5名 → 11.5名
- 生産性: ×1.0 → ×1.25
- 組織換算: 7.5 → 14.4 (**1.9倍**)

## 10. 最終定義

> **人材を変えるのではない。構造を変える。**

| 従来マネジメント | 本設計 |
|---|---|
| 数値と行動の管理 | 意味・成長・心理の構造管理 |
| 個人の責任として処理 | 構造の問題として設計で解決 |
| 成果が出ない=能力の問題 | 成果が出ない=構造の不備 |
| 離職=失敗・損失 | 離職=構造改善資産 |
| 知識は個人に閉じる | 知識は組織資産として蓄積 |

> 営業は商品を売る行為ではない。自己を伝達する行為である。
> 最終的に選ばれるのは商品ではなく、納得であり、その納得は提案者から生まれる。
> **自己を売れる人材を、構造によって再現性をもって量産する。**
