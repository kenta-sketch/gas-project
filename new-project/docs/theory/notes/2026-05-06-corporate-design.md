# QUAD MIND 企業向け 完全設計書

> 「誰を営業に置くか」ではなく
> 「誰を、どこに、どう置けば売れるか」を逆算する。

## 1. なぜ今、これが必要か

営業組織が伸び悩む本当の理由は採用の失敗でも教育の不足でもない。**最大の原因は「配置の設計がない」こと**。

| ミスの種類 | 起きていること | 結果 |
|---|---|---|
| 配置の誤り | Aが低い人を新規開拓に置く / Cが低い人をクロージングに置く | 初動で止まる・案件が流れる・離職 |
| 育成の誤り | 気合いと経験則で全員を同じように育てる | 伸びる人と潰れる人が属人的に決まる |
| 評価の誤り | 総合印象・面接の上手さで人材を見る | 口がうまい人が採用される・本当の適性を見逃す |

## 営業成果の式

> **営業成果 = A(行動量) × B(信頼形成) × C(決断誘導) × D(再現性)**

掛け算であるから、ゼロに近い値が1つあれば全体が崩れる。

## 営業を4工程に分解する

| 工程 | 営業種別 | 求められる能力 | 主エンジン |
|---|---|---|---|
| ① | 接触営業 | 声かけ・テレアポ・飛び込み。断られても止まらない初動力 | A(突破力) |
| ② | 信頼形成営業 | ヒアリング・接客・既存対応。「この人から買う」と思わせる力 | B(共感・信頼) |
| ③ | クロージング営業 | 提案・比較・決断促進・ネガ処理。顧客の迷いを整理し決断まで導く | C(直観的判断) |
| ④ | 再現営業 | データ管理・改善・教育・マニュアル化。売れた理由を再現可能に | D(再現・管理) |

## 評価構造と測定設計

各エンジン25点満点、合計100点。**重要なのは総合点ではなく、A/B/C/Dの内訳**。

### 測定の4素材

| 評価素材 | 比重目安 | 何を見るか |
|---|---|---|
| 面接質問 | 20% | 思考傾向・自己認識・エピソードの有無 |
| ロープレ | 30% | 実際の反応・営業行動・咄嗟の判断 |
| 行動ログ | 30% | 日々の実績・継続性・数字への態度 |
| 第三者評価 | 20% | 上司・顧客・同僚からの信頼度 |

### 点数帯の定義(各エンジン25点満点)

| 点数帯 | 評価 | 実務判断 |
|---|---|---|
| 0〜9 | 補助が必要 | その領域を主戦場にすると高確率で苦戦 |
| 10〜15 | 標準領域 | 型があれば実務可能。強みにはなりにくい |
| 16〜20 | 実務戦力 | 配置次第で成果に直結。2領域以上で高成果 |
| 21〜25 | 武器領域 | 明確な強み。ここを活かさない配置は人材の無駄 |

### 合計点の概観

| 合計点 | 企業判断 |
|---|---|
| 0〜39 | 補助・事務・バックオフィスが安定 |
| 40〜59 | 育成前提。型の注入で戦力化 |
| 60〜74 | 標準的営業戦力。内訳次第で成果が大きく変わる |
| 75〜84 | 高適性。2領域以上が強く即戦力 |
| 85〜94 | 上位人材・管理候補 |
| 95〜100 | 統合型。営業戦略・組織設計まで担える希少人材 |

## A: 動物的感情 - 行動量・突破力・初動

### Aを見るルーブリック

| 項目 | 1点(要補強) | 3点(標準) | 5点(武器) |
|---|---|---|---|
| 初動の速さ | 指示待ち | 指示があれば動く | 言われる前に動く |
| 接触力 | 自分から声をかけない | 必要最低限はかける | 自然に新規接触を作る |
| 拒絶耐性 | 断られると止まる | 少し落ちるが戻れる | 切り替えが速い |
| 行動量 | 標準以下 | 標準量 | 周囲より明確に多い |
| 場を動かす熱量 | 場に飲まれる | 場に合わせる | 場の空気を変えられる |

### Aを見る面接質問
- 初対面の人に話しかけることに抵抗はありますか?(エピソード重視)
- 断られた時、次にどう動きますか?
- 過去に、自分から動いて成果を出した経験はありますか?
- 量を求められる仕事に対して、どう感じますか?

### Aの点数帯別 企業判断

| A点数 | 実務判断 |
|---|---|
| 0〜9 | 新規開拓・飛び込み・テレアポに置くと初動で止まる。受付型・既存対応が現実的 |
| 10〜15 | 指示があれば動ける。量を求めすぎると潰れる |
| 16〜20 | 新規接触・イベント営業・アポ獲得に投入可能 |
| 21〜25 | 新規・立ち上げ・即決営業の起爆剤(制御設計必須) |

## B: 機械的感情 - 共感・信頼形成・関係維持

### Bを見るルーブリック

| 項目 | 1点 | 3点 | 5点 |
|---|---|---|---|
| 傾聴力 | 話を遮る | 最低限聞ける | 本音まで引き出す |
| 共感力 | 一方通行 | 表面的に合わせる | 不安・迷いを言語化できる |
| 安心感 | 圧がある・雑 | 普通に接客 | 「この人なら大丈夫」と思わせる |
| 空気把握 | 違和感に気づかない | 明確な反応には気づく | 表情・沈黙・声色から察知 |
| 関係維持 | その場限り | 必要な連絡はできる | 紹介・深い信頼に繋げる |

### Bを見る面接質問
- 人から相談されることは多いですか?(頻度・内容・その後)
- お客様が不安そうな時、どこを見ますか?
- 苦手な人と関わる時、何を意識しますか?
- 信頼される人と、ただ話しやすい人の違いは?

> 見るポイント: **自分がどう見られたいかだけを語る人はBが高いとは言い切れない**。相手目線・相手の不安から語れているかを見る。

### Bの点数帯別 企業判断

| B点数 | 実務判断 |
|---|---|
| 0〜9 | 顧客接点中心はクレームリスク高。補助・裏方・数字管理へ |
| 10〜15 | 定型接客は可能。深い信頼形成は弱い |
| 16〜20 | 既存営業・接客・紹介獲得で安定成果 |
| 21〜25 | 顧客・チームから信頼される。高単価・法人・マネジメント |

## C: 動物的理性 - 本質把握・提案・クロージング

### Cを見るルーブリック

| 項目 | 1点 | 3点 | 5点 |
|---|---|---|---|
| 本質把握 | 表面的 | ある程度整理 | 本当の迷いを見抜く |
| 提案力 | 商品説明で終わる | 顧客に合わせて提案 | リアルタイムで組み替える |
| 比較力 | 競合比較に弱い | 基本比較できる | 価値・リスク・条件を整理 |
| ネガ処理 | 反論されると黙る | 一般的な切り返し | 不安の根本を解消 |
| 決断誘導 | 「検討してください」 | 背中を押せる時もある | 顧客が納得して決める状態を作る |

### Cを見る面接質問
- お客様が「検討します」と言った時、何を確認しますか?
- 売れなかった時、どこを振り返りますか?
- 顧客が決められない理由は、何だと思いますか?

> 見るポイント: 「頑張って説明します」ではCは高くない。**「なぜ決められないのか」を構造で分解できる人**がC高得点者。

### Cの点数帯別 企業判断

| C点数 | 実務判断 |
|---|---|
| 0〜9 | クロージングに置くと案件が流れる |
| 10〜15 | 提案はできるが決定力は弱い |
| 16〜20 | クロージング・提案営業に投入可能 |
| 21〜25 | トップ営業候補。営業教育担当へ |

---

## D: 機械的理性 - 再現性・管理・仕組み化

### Dを見るルーブリック

| 項目 | 1点 | 3点 | 5点 |
|---|---|---|---|
| 数字管理 | 感覚で動く | 基本数字は追える | 数字から改善点を出せる |
| 記録力 | 行動が残らない | 指示があれば記録 | 自分から記録し資産化 |
| 分析力 | 感覚で振り返る | ある程度整理 | 売れた理由・売れない理由を分解 |
| 仕組み化 | 属人的 | 一部マニュアル化 | 誰でも使える型に変換 |
| 教育展開 | 教えるのが苦手 | 自分のやり方は伝えられる | 相手に合わせて再現させられる |

### Dを見る面接質問
- 自分の成果をどう管理していますか?(具体性)
- 売れた理由を人に説明できますか?
- 数字が落ちた時、どこから確認しますか?

### Dの点数帯別 企業判断

| D点数 | 実務判断 |
|---|---|
| 0〜9 | 感覚営業。組織資産化しない。管理者には置かない |
| 10〜15 | 現場営業として問題なし。管理者には数字教育必要 |
| 16〜20 | 営業管理・教育補助・法人営業で高い価値 |
| 21〜25 | 組織設計人材。KPI設計・教育制度構築まで |

## 9. 点数の変動可能性

| エンジン | 変動性 | 理由 | 伸ばす方法 |
|---|---|---|---|
| A | 低〜中 | 先天性・身体性が強い | 行動ハードル下げる・成功体験・ペア営業 |
| B | 中 | 経験・クレーム対応で伸びる | 傾聴訓練・顧客心理共有 |
| C | 中〜高 | 実戦と振り返りで伸びる | クロージングロープレ・失注分析 |
| D | 高 | 教育と仕組みで最も伸ばしやすい | 日報・KPI・マニュアル・週次レビュー |

> **育成投資の順番: D → C → B → A**
> Dは仕組みで伸ばせる。Cは実戦で伸びる。Bは経験で伸びる。
> **Aは採用時点で見ておくべき**(後から大幅変更は現実的でない)。

### 現実的な伸び幅目安

| 初期点数 | A | B | C | D |
|---|---|---|---|---|
| 0〜9 | +1〜3 | +2〜4 | +2〜5 | +3〜6 |
| 10〜15 | +2〜4 | +3〜6 | +4〜7 | +5〜10 |
| 16〜20 | +2〜5 | +2〜5 | +3〜6 | +3〜7 |
| 21〜25 | 維持 | 精度向上 | 言語化・高度化 | 管理者化 |

## 11. 採用と育成の使い分け

### 採用時に重視すべきエンジン

| 職種 | 重視順 | 理由 |
|---|---|---|
| 新規営業 | A → C → B | Aは後から変えにくい |
| 接客・店舗営業 | B → C → A | Bが低いと顧客の不安を拾えない |
| 法人営業 | C → D → B | 長期案件の構造理解 |
| 既存営業 | B → D → C | 信頼の継続と情報管理 |
| 管理者候補 | D → B → C | Dだけ高くBが低いと冷たい管理者 |
| 新規事業責任者 | A → C → D | 立ち上げの熱量と構造設計 |

### 点数更新のタイミング

| タイミング | 目的 |
|---|---|
| 採用時 | 初期配置の判断基準 |
| 1ヶ月後 | 現場適応を確認 |
| 3ヶ月後 | 最も現実的に適性を見やすい。本配置の見直し |
| 6ヶ月後 | 育成効果の確認 |
| 12ヶ月後 | 配置転換・昇格・管理者選抜 |

## 12. 倫理的配慮と理論の限界

- 本理論は「欠陥の特定」ではなく「発達支援と配置設計」のために使用する
- 特定エンジンが低いことを「能力の欠如」として固定的に扱わない
- 単発診断ではなく、継続的フィードバックとの組み合わせで運用する
- 個人責任化ではなく、環境調整との併用を基本とする
- エンジン強度は「状態 × 特性 × 状況適合」の三層で扱う

## 結論

| 従来の問い | クアッドマインドが与える問い |
|---|---|
| この人は営業に向いているか? | この人はどの営業工程で利益を生むか? |
| 総合点・印象で評価 | A/B/C/Dの内訳で配置を逆算 |
| 気合いと経験で同じように育てる | 弱いエンジンに対応した型の注入 |
| トップ営業の感覚を共有 | 再現性を設計して組織資産にする(D) |
| 3ヶ月様子を見てから判断 | 採用段階で配置工程を決め、逆算して採る |

> Aは命。Bは社会。Cは経験。Dは構造。
> Observerは自由意志。
> **営業適性は才能ではない。配置の設計問題である。**
