# 職種別クアッドマインド必要値のメタ分析

> 73職種について、A/B/C/Dの推奨レンジを厚生労働省 [shigoto.mhlw.go.jp](https://shigoto.mhlw.go.jp/User/) と ONET OnLine の職務・能力データ、日本の資格・職能団体の公式説明を突き合わせて外挿した分析。

各軸25点満点。読み方: 22-25が中核、18-21が強い補助、13-17があると有利、4-12は補助的。

## 4軸への変換指針

- **A**: Initiative や不確実な場面で前に出る推進力
- **B**: Empathy や Service Orientation のような対人支援・関係維持
- **C**: Complex Problem Solving や Systems Analysis のような診断・解釈・判断
- **D**: Attention to Detail や記録・手順・基準順守への適合性

## 横断パターン(実務ルール)

シンプルな整理:
- **新規開拓・現場即応・身体リスク・創作の立ち上がり** → A高
- **継続支援・ケア・信頼形成・利害調整** → B高
- **診断・解釈・提案・専門判断** → C高
- **監査・法規・安全・記録・品質・手順** → D高

> 現代の知的職務は、営業・マーケ・IT・IR・CRC・施工管理まで含めて、**C/Dの不足が成果不安定**につながる。

### 採用や配置の核心

> 合計点ではなく**「形」を見る**こと。法人営業と経理は総量が近く見えても求める形は違う。
> 実務ではまず**主軸を2つ決め、その2軸だけは下限を割らせない**運用が最も実用的。

### 補正の目安

| 実務条件 | 補正 |
|---|---|
| 新規開拓比率が高い | A +2〜4 |
| 継続利用・解約抑止・ケアが重要 | B +2〜4 |
| 高額提案・診断・専門判断が重要 | C +2〜4 |
| 監査・法規・安全・帳票・品質が重い | D +2〜4 |
| 役職者・リーダー職 | B +2、D +2、必要に応じC +1〜2 |

> このテーブルは「向き不向きの断定表」ではなく、**「成果が出やすい職務要求の地図」**。
>
> Aが低い人でも経理や校閲では十分強く、Bが低い人でもDBや品質では戦える。
>
> 致命傷になる職種:
> - **B不足が致命**: 接客・看護・CS・人事・産業カウンセラー
> - **C不足が危険**: 法務・医師・データサイエンス・樹木医・ホワイトハッカー
> - **D不足が事故**: 経理・品質・CRC・点字翻訳・校正校閲
