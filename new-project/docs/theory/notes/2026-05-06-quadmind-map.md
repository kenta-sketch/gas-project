# クアッドマインド 職種適性マップ(精密版)

## 判定式

> **職種適性 = 主エンジン適合度 × 副エンジン補完度 × 最低ライン通過 × 過剰リスクの少なさ**

## 配置してはいけない危険パターン

| パターン | 結果 |
|---|---|
| **A低 × 新規営業・新規事業** | 動き出しで止まる。努力不足ではなく**配置ミス** |
| **B低 × 人事・接客・管理職** | 人が離れる。顧客・候補者・部下との摩擦増 |
| **C低 × 企画・法務・営業判断** | 判断が浅い。論点がズレる。決め切れない |
| **D低 × 経理・法務・管理職・教育** | 組織が崩れる。記録・正確性・再現性に支障 |
| **A高すぎ × 経理・品質・法務** | 衝動性が邪魔。安定処理・慎重判断が必要 |
| **B高すぎ × 厳格判断職** | 人に寄りすぎる。法務・監査・品質では基準優先の場面がある |
