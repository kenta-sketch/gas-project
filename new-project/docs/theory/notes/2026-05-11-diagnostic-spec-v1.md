> 質問数を増やすことがクオリティを上げるのではない。

## G2: A発火 / A表出の分離測定

### 内的A測定(感情の強さ)5問

「一人でいるとき」「見られていないとき」に限定 → Bの影響を排除。

| iA-5 | 感情が動いていないとき、何をしても平坦な感じがする | 補助 | ×1.0 |

### 表出A測定(外に出すか)5問

| ID | 質問文 | 種別 | 重み |
|---|---|---|---|
| eA-1 | 感じていることを、相手に伝えることができる | コア | ×1.5 |
| eA-2 | 嬉しいときに、素直に喜びを表現できる | コア | ×1.5 |

### 追加分岐: A凍結型 vs A抑圧型(2問)

内的A高 × 表出A低 の場合のみ表示。

| ID | 質問文 | 判定 |
|---|---|---|
| FZ-1 | 強いストレス下で、何も感じなくなる・頭が真っ白になることがある | 凍結判定 |
| FZ-2 | 過去に強い恐怖や衝撃的な経験があり、今も特定の状況で固まる感覚がある | 凍結判定 |

### 4種判別ロジック

| 内的A | 表出A | 判定タイプ | 介入方針 |
|---|---|---|---|
| 低(<13) | 低(<13) | 真性A低 | 着火体験の設計。小さな「本気」を経験させる |
| 高(≥13) | 低(<13) | A抑圧型 / A凍結型※ | FZ質問で分岐。安全な表出環境 or 専門的支援 |
| 高(≥13) | 高(≥13) | A管理型(統合寄り) | 介入不要。強みとして活用 |
| 低(<13) | 高(≥13) | ⚠ 演技的表出フラグ | 要確認。自己申告バイアスの可能性 |

**A凍結型への対応(重要)**:
- FZ-1・FZ-2が両方Yesに近い → A凍結型フラグ + 「専門的なサポートを検討することをお勧めします」と表示。診断ツールが直接介入しない。
- いずれかがNoに近い → A抑圧型として処理。

## G4: 統合状態の直接検出

> 「全軸15以上・最大最小差8未満」では偽の中庸と本物の統合を区別できない。
> 静的スコアではなく **「切り替えの自覚」を直接問う質問群** を追加。

### Observer起動測定(5問)

| ID | 質問文 | 重み |
|---|---|---|
| OB-1 | 感情が強く動いたとき、少し間を置いてから行動できることがある | ×1.5 |
| OB-2 | 「今、自分は感情的になっている」と、感情の中で気づけることがある | ×1.5 |
| OB-3 | 衝動的に動きそうになったとき、一瞬止まれることがある | ×1.0 |
| OB-4 | 行動した後で「あのときAで動いていた」と振り返れることがある | ×1.0 |
| OB-5 | 「本当はこうしたかった」と後から気づくことが、最近減ってきた | ×1.0 |

### 切り替え自覚測定(5問)

| ID | 質問文 | 重み |
|---|---|---|
| SW-1 | 論理で考えるモードと、感覚で動くモードの両方を、自分の中に感じる | ×1.5 |
| SW-2 | 状況によって、自分の動き方が変わることを自覚している | ×1.5 |
| SW-3 | 「今は感情より論理で判断すべき場面だ」と判断できることがある | ×1.0 |
| SW-4 | 「今は分析より直感を信じていい」と思えることがある | ×1.0 |
| SW-5 | 自分の反応パターンを、ある程度予測できる | ×1.0 |

### 判定式と出力ロジック

```
統合指数 = (Observer得点 + 切り替え得点) ÷ 2
```

| 統合指数 | 全軸スコア条件 | 判定 | 出力方針 |
|---|---|---|---|
| ≥20 | 全軸≥13 | 本物の統合(Lv.2〜3) | 統合型プロファイルを表示 |
| ≥20 | 一軸でも<10 | 部分統合 | 低軸を「成長課題」として表示 |
| <15 | 全軸均衡 | ⚠ 偽の中庸フラグ | 再診断を促す文言を表示 |
| <15 | 軸間差大 | 単独運転状態 | 主軸エンジンの診断を優先 |

偽の中庸フラグ時の出力文:
> 「あなたのスコアは全体的に均衡していますが、強い切り替え体験の自覚が低い傾向があります。各軸をより具体的な場面で振り返ると、より正確な結果が得られます。」

## G3: 責任感の3形態分離

> 「責任感がある」は採用現場で最も使われる言葉だが最も内実が空洞だ。
> D型の人にA型責任感を求める組織、B型の人をリーダーに置く組織̶̶この不一致が離職とバーンアウトの温床。

### D型責任感(規範・ルール準拠)4問

| ID | 質問文 |
|---|---|
| DR-1 | ルールや約束を守ることは、状況に関わらず重要だと思う |
| DR-2 | 決めたことを最後まで実行することが、責任だと考える |
| DR-3 | 手順やプロセスが正しければ、結果が予想外でも問題ないと思える |
| DR-4 | 組織のルールに従うことが、自分の役割だと感じる |

### B型責任感(他者期待への応答)4問

| ID | 質問文 |
|---|---|
| BR-1 | 頼まれたことは、できる限り断らないようにしている |
| BR-2 | 期待されていると感じると、それに応えなければという気持ちが強くなる |
| BR-3 | 自分の仕事が誰かの役に立っていると感じると、頑張れる |
| BR-4 | 失望させることへの恐れが、行動の動機になっていることがある |

### A型責任感(結果・自己感情への準拠)4問

| ID | 質問文 |
|---|---|
| AR-1 | 自分がやると決めたことは、誰に言われなくてもやり切る |
| AR-2 | 成果が出なければ、ルールを守っていても意味がないと思う |
| AR-3 | 自分が「正しい」と思えないことには、責任を持てない |
| AR-4 | 責任を感じるのは、自分が心から関わりたいと思っている仕事だ |

### 採点ロジック

```
各型スコア(4〜20点)を算出 → 最高得点の型を「主責任感タイプ」として出力
2位との差が3点未満 → 「複合型」として両方を記述
```

出力例: 「あなたの責任感はB型主軸(期待応答型)です。誰かに必要とされることが行動エネルギーになりますが、期待が重なると抱え込みすぎるリスクがあります。」

## G5: 組織毀損プロファイルの検出

### 🚨 出力設計の原則(重要)

> **内部用途(採用担当・マネジャー向け)と外部出力(本人向け)を必ず分離**する。
> 本人向け出力では「**癌候補**」という表現を一切使わない。傾向と環境の話として記述する。

### 類型①: 承認略奪型(B高+他者攻撃指向)

表面は協調的。水面下で情報操作・評判毀損・功績の横取り。Bは自己防衛に使われている。

| ID | 質問文 |
|---|---|
| AG-1 | 自分の成果が正当に評価されていないと感じることが多い |
| AG-2 | うまくいかないとき、周囲や環境のせいだと感じることがある |
| AG-3 | 自分より評価されている人を見ると、腑に落ちないことがある |

### 類型②: ルール暴力型(D高+正論支配)

ルールと論理は正しいが、それを人への攻撃に使う。組織の創造性と心理的安全性を構造的に破壊。

| ID | 質問文 |
|---|---|
| RV-1 | 感情論で話す人と議論するのは、時間の無駄だと思う |
| RV-2 | ルールや論理に従わない人には、はっきり指摘すべきだと思う |
| RV-3 | 正しいことを言っているのに受け入れられないとき、相手の理解力を疑う |

### 類型③: 衝動暴走型(A未統合+感情爆発)

衝動がそのまま行動になる。マネジャー職についていると最も被害が大きい。

| ID | 質問文 |
|---|---|
| IM-1 | 怒りや不満が、思っていたより強く出て後悔することがある |
| IM-2 | 気分によって、同じ人・同じ仕事への態度が大きく変わることがある |
| IM-3 | 感情が高ぶると、言ってはいけないことを言ってしまうことがある |

### 介入指針(類型別)

| 類型 | 内部出力(管理職向け) | 介入指針 |
|---|---|---|
| 承認略奪型 | 承認欲求の処理パターンに注意が必要な傾向。面接での深掘りを推奨。 | 個人成果が見えやすいポジションへ配置+コーチング。改善しない場合は役割の再設計。 |
| ルール暴力型 | 正論による場の支配傾向。チームの心理的安全性への影響を確認。 | 「正しさより関係性」の経験を積ませる。改善しない場合は管理職から外す。 |
| 衝動暴走型 | 感情の波が大きく、チームの予測可能性に影響するリスクがある。 | Observer訓練+身体状態管理(睡眠・疲労)の構造的確保。人の管理権限を段階的に付与。 |

## 品質向上: 3つの構造的改善

### 改善①: 質問形式の変更(即日実装可能)

**変更前(リッカート形式)**:
「感情が動いたとき、少し間を置いてから行動できる」→ 1〜5で評価

**変更後(状況固定形式)**:
「重要な会議で反論したいと思ったとき、あなたは?」
- a) その場で言う
- b) 言い方を考えてから言う
- c) 後で個別に伝える
- d) 結局言わないことが多い

→ 場面を具体的に固定することで「理想の自分」への逃げ道が塞がれる。測定精度が上がる。

### 改善②: 複数回答の時系列設計

人間のA/B/C/Dは状態によって変動する。一時点のスナップショットはその日のコンディションを測っている可能性。

**設計**: 2週間おきに3回受診 → 平均スコアと変動幅を出力
- 変動幅が大きい人 → 「状態依存型」(環境に強く影響される)
- 変動幅が小さい人 → 「特性が固定している」(安定しているが更新されにくい)

### 改善③: 行動ログとの照合(診断→介入ツールへの転換)

診断後に「今週、この場面でどう動いたか」を短く記録させる。週に一回、3問だけ。

**週次ログ質問(3問)**:
1. 今週、言いたいことを言わなかった場面はありましたか
2. 今週、感情が先に出て後悔した場面はありましたか
3. 今週、直感が当たった場面はありましたか

4〜8週間蓄積 → 自己申告スコアと行動ログの一致度を表示
- 一致している人 → 「診断精度が高い」フィードバック
- 乖離している人 → 「あなたのスコアと実際の行動にズレがあります」→ Observer訓練の入口

→ これにより診断が **「測定ツール」から「成長ツール」に転換** する。
